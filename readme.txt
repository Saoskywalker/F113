2021.9.23
1.修改程序，修改内容为增加上电开启PMOS的延时时间，由之前的15ms延时修改为45ms时间，
用以解决某些板子上电开机延时问题。

2021.9.21
1.修改程序，修改内容为增加看门狗功能  防止程序复位失败出现上电之后不成功的情况。
2.修改MCU的主频由之前的16MHZ 修改为4MHZ.   MCU的工作电压范围为1.8V-5.5V.  但是设定的
MCU复位电压为2.6V。
3.基于以上修改之后做的测试发现，前面低温测试出现自动跳档的情况不再出现，可以解决由于
mcu复位不成功或者异常上电程序出现读取MCU内部EEPROM存储的档位信息错误的情况出现。
以上修改2点内容只要目的是为了保证MCU 正常复位，保证在供电不正常的情况下正常开机。

2021.9.15
1.修改客户反馈的触摸问题，问题描述为，同时触摸2个按键，出现按键功能混乱情况。
2.修改开启马达以及关闭马达的控制时间流程。
开启马达的流程为，先开启MOS管，10MS之后开启PWM输出。
关闭马达的流程为，先关闭PWM输出，10ms之后关闭MOS管。
